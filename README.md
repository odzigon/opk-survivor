## // BRIEFING

Le signal NEXIUM est en ligne. Des unités hostiles convergent sans relâche vers votre position, et chaque seconde de survie aggrave la menace. Votre arme tire seule : votre mission consiste à tenir le terrain, esquiver, monter en puissance et abattre les boss qui se dressent sur votre route.

À chaque run, vos choix de montée en niveau façonnent un build unique parmi un arsenal d'armes, de passifs et d'évolutions. Entre deux tentatives, l'or récolté alimente une boutique permanente et une méta-progression qui rendent chaque nouveau déploiement plus solide que le précédent.

## // FONCTIONNALITÉS

- Vagues d'ennemis infinies, avec un bestiaire varié et des boss aux attaques distinctes
- Tir automatique — visée libre (souris / joystick droit) ou ciblage automatique de l'ennemi le plus proche
- Système de niveaux et d'XP, avec choix d'amélioration à chaque montée de niveau (relance, passe contre de l'or, ou bannissement d'une option)
- Arsenal d'armes évolutives pouvant fusionner en version ultime avec le bon passif
- Passifs cumulables pour personnaliser chaque run
- Opérateurs multiples, chacun avec un rôle propre et une capacité active dédiée
- Boutique permanente : bonus durables et déblocages (opérateurs, armes, capacités)
- Méta-progression persistante, avec système de prestige
- Quêtes, succès, palmarès et malédictions (modificateurs risque / récompense)
- Modes Sans fin, Boss Rush et Quotidien, avec plusieurs paliers de difficulté
- Coopération multijoueur en ligne (salon avec code de partie)
- Menu complet : réglages, aide intégrée, journal des versions, connexion Discord
- Sauvegarde locale automatique, avec option de sauvegarde cloud
- Compatible clavier, souris et écran tactile (mobile)

## // CONTRÔLES

### Clavier & souris

| Action | Touche / Bouton |
|---|---|
| Déplacement | `WASD` / `ZQSD` / flèches |
| Visée (optionnelle) | Souris |
| Tir | Automatique |
| Capacité (ex. dash) | `Espace` / `Maj` |
| Pause | `Échap` / `P` |

### Mobile / Tactile

| Action | Geste |
|---|---|
| Déplacement | Joystick virtuel gauche (apparaît au toucher) |
| Visée (optionnelle) | Joystick virtuel droit |
| Tir | Automatique |
| Capacité | Bouton dédié |

> Sans visée manuelle activée, l'ennemi le plus proche est ciblé automatiquement : le jeu reste jouable à une seule main.

## // MODES DE JEU

- **Sans fin** — survie classique face à des vagues infinies
- **Boss Rush** — une succession de boss à affronter, sans les vagues classiques
- **Quotidien** — run avec conditions imposées, partagée par tous les joueurs pendant 24h

Chaque mode peut être abordé à différents paliers de difficulté : **Recrue**, **Standard**, **Vétéran**, **Cauchemar**.

En complément, un mode **Coopératif / Multijoueur en ligne** permet d'affronter les vagues avec d'autres joueurs via un salon (création ou code de partie).

## // PROGRESSION

### Pendant une run
- **XP** récoltée sur les ennemis pour monter de niveau
- À chaque niveau, choix d'une amélioration (nouvelle arme, montée de niveau d'arme ou passif) — possibilité de **relancer** les propositions, de **passer** une amélioration pour gagner de l'or à la place, ou de **bannir** une option indésirable (nombre limité par run)
- **Évolutions** : une arme au niveau maximum associée au bon passif se transforme en version ultime
- **Malédictions** : modificateurs optionnels qui augmentent la difficulté en échange de meilleures récompenses

<details>
<summary>Voir la liste des malédictions</summary>
<br>

| Malédiction | Effet |
|---|---|
| Fragilité | PV max réduits de 30 % |
| Hémophilie | Aucune régénération |
| Marée | +60 % d'ennemis |
| Titans | Ennemis +55 % PV |
| Furie | Ennemis +30 % dégâts |
| Glas | Les boss arrivent plus tôt |
| Myopie | Rayon de ramassage réduit |
| Disette | −25 % d'XP gagnée |

</details>

### Entre les runs
- **Or** conservé d'une partie à l'autre, à dépenser dans la **boutique permanente**
- **Quêtes** et **succès** avec récompenses dédiées (nouvelles armes, opérateurs, capacités ou or)
- **Palmarès** des meilleurs scores et runs, avec système de **prestige**
- **Méta-progression** : l'ensemble de vos déblocages persiste durablement
- **Connexion Discord** pour sauvegarder sa progression dans le cloud
